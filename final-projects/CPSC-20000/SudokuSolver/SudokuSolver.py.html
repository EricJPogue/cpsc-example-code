Team delivering application & update:

This final project will be done in association of team CashMoney. However, this final project will be done by myself. This project was last updated during Sprint 8, May 3rd, 2021.

End User Perspective & languages used:

In this program, you will see a generated Sudoku puzzle that will be solved by the program that will mimic the thought process of a person solving the puzzle, mainly as a check and go using back tracking verifitcation functions. Originally, I had intentions to have a visual heavy projects with CarCrash, but I could not seem to figure out how to translate glowscript code to normal python commands were different and glowscript runs off Vpython (Visual python) code.

Python will be the star of the show in this final project, but will utilize imports
