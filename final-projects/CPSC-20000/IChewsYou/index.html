What is ..

Hi, and welcome to I Chews You! This app helps you meet your four-legged best friend. To operate the app, the user must answer a series of questions that will later match them with a dog that best suits their wants and lifestyle.

List of desied features

  • bio of each dog
  • displays top match
  • compatibility with different breeds

Non-prioritzed list of desired features

  • implement a location where you can get the dog
  • dog breed personality description

Languages and Platforms

Features We Have Learned

Beyond Knowledge Features
