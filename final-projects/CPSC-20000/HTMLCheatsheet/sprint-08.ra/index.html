HTML: A Beginner's Cheatsheet to Creating a Web App

I wanted to create this website for those who have never made a web app using HTML before. I can relate to those who are intimidated by using coding for the first time. I had to always ask for help and have several tabs open on my browser to find what I was looking for. Here is a little cheatsheet for your convience. You can use Visual Studio Code or Notepad (for Windows). My examples will be done in Windows Notepad. Please note that this web app will go over the basics on how to create a web app.

Areas to Cover

  • How to start and close the file
  • Headings and Paragraphs
  • Links and Images
  • Organization
  • Styling

Key Notes

  • Save your file OFTEN.
  • Reload your Web app to see changes immediately after saving the HTML file.
  • Save an extra copy of the file for emergencies.
  • If you cannot figure out where something went wrong, look online or ask a friend for a pair of fresh eyes.
  • Stay calm. I know it can be very intimidating, but you will learn!
